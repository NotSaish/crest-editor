# ⚡ Crest Editor

A lightweight, modern code editor built with **Jetpack Compose** for Android.

Crest Editor is a fast, offline-first coding environment designed for mobile developers who want a clean and powerful coding experience on Android.

---

## ✨ Features

- 🧠 Custom Compose-based editor engine (no WebView)
- 🎨 Syntax highlighting for:
  - Python
  - Kotlin
  - Java
  - JavaScript / TypeScript
  - C / C++
- 📁 Built-in file manager
- 💾 Save / unsaved indicator system
- 🌌 Dark modern UI theme
- 🔢 Line numbers support
- ⚡ Smooth scrolling experience
- 📱 Fully offline

---

## 🏗️ Architecture

Crest Editor is built using:

- Jetpack Compose UI
- Kotlin Coroutines for async file handling
- Custom text rendering system using `BasicTextField`
- Flow-based state management
